### 默认
```js
<div>
  <div style={{display: 'flex', alignItems: 'center'}}>
    <Avatar size="large"/>
    <Avatar/>
    <Avatar size="small"/>
  </div>
  <div style={{display: 'flex', alignItems: 'center'}}>
    <Avatar size="large" shape="square" style={{backgroundColor: '#f06292', color: '#ffebee'}}/>
    <Avatar shape="square"/>
    <Avatar size="small" shape="square"/>
  </div>
</div>
```
