# Yoshino
Yoshino, 一个基于react的pc端组件库，名字源自轻小说作品《约会大作战》及其衍生作品中登场的人物。长相宛若法国娃娃般美丽的蓝发少女，有着梦幻般的外貌和娇小的身材。第二精灵，识别名为〈隐居者（Hermit）〉。生性温驯而胆小怕生，几乎不敢与人直接对话。左手拿着的兔子型的手偶四糸奈（よしのん）是独立的人格，也是她最好的朋友。
